## Краткое описание алгоритма
### **Основная функция `adaptivePiyavsky`**
- **Входные параметры:** функция, границы [a,b], точность eps, максимум итераций

- **Выходные данные:** результат оптимизации с найденным минимумом

### **Ключевые этапы алгоритма:**
1. Оценка константы Липшица - `estimateLipschitzConstant`

2. Инициализация начальными точками a и b

3. Основной цикл оптимизации:
-
  - Поиск интервала с наилучшим потенциалом
  - Вычисление функции в середине интервала
  - Добавление новой точки с сохранением порядка
  - Проверка условия остановки

### **Вспомогательная функция `estimateLipschitzConstant`**

- Вычисляет максимальный наклон функции на равномерной сетке
- Добавляет запас 20% для надежности

### **Критерии остановки:**
- Достигнута заданная точность eps
- Превышено максимальное число итераций

### **Теоретическое обоснование:**
- Алгоритм использует "зубчатую верхнюю оценку" (sawtooth upper bound)
- Гарантирует нахождение глобального минимума для липшицевых функций
- Основан на теореме Пиявского

### **Преимущества:**
- Не требует вычисления производных
- Устойчив к наличию множества локальных минимумов
- Гарантированная сходимость
